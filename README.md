# Examen Mercadolibre
ApiGeoMeli es una aplicación con el objetivo principal de poder obtener información desde una dirección ip de cualquier parte del mundo. Cuenta con el historial de consultas realizadas para visualizar que paises fueron los mas consultados, como también conocer cual fue el país más lejano, el mas cercano y un promedio de todas las distancias  de direcciones consultadas respecto a Argentina.<br>
Para poder utlizar ApiGeoMeli, deberá descargar este repositorio y desde una consola bash ejecutar `./start.sh`<br>
(Serà necesario contar con Maven previamente instalado para su correcta ejecución).

Por último, abrir el navegador e ingresar a `http://localhost:8080`.
